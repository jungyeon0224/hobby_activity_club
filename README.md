# Hobby Activity Club (2021145050 유정연)

Flutter 기반의 취미 공유 플랫폼 앱입니다.

## 주요 기능
- 취미 리스트 확인 및 상세 페이지 보기
- 신청 폼 작성 (이름, 전화번호 입력)
- 마이페이지에서 후기 작성 및 목록 보기
- 참여 인원수 표시
- 후기 수정 및 삭제 기능

## 실행 방법
flutter pub get
flutter run
